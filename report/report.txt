Procrastination Risk Detector
A Simple Tool to Analyze Study Habits


1. Introduction

Procrastination is something almost every student faces. We make plans to study but then end up scrolling on our phones or watching YouTube instead. I personally have faced this many times, and I thought it would be interesting to make a simple program that can help us see if we are actually following our study plans or not.

This project is a basic program called Procrastination Risk Detector. The idea is simple – you enter your planned study hours, what you actually studied, and how many distractions you had, and the program tells you whether you’re at low, moderate, or high risk of procrastinating.

2. Problem Statement

Students often create study schedules but don’t follow them. Because of this, they end up wasting time and not completing their work on time. One of the main reasons is procrastination, which happens when we get distracted easily or just delay our tasks.

There should be a way to help students realize if they are procrastinating, so they can try to fix it. This program is a small step toward that.

3. Objective

The main goals of this project are:

- Build a simple program that checks study habits.
- Compare planned study hours with actual hours.
- Count how many distractions happened during study.
- Decide the level of procrastination risk.
- Give the user a clear output and a little tip to improve.

4. Tools Used

For this project, I used:

- Python – because it’s easy to learn and perfect for small programs like this.
- GitHub – to store the code and keep track of changes.
- VS Code – as the code editor to write and test the program.

5. How the Program Works

The program takes three inputs from the user:

1. Planned study hours
2. Actual study hours
3. Number of distractions

Then it uses some if-elif-else conditions to decide the risk level:

- If actual hours are equal to or more than planned and distractions are 1 or less → Low Risk
- Else if actual hours are at least half of planned → Moderate Risk
- Otherwise → High Risk

Finally, it prints a small report showing the inputs and the result, plus a motivational tip.

6. Features

- Easy to use – just type in your numbers.
- Compares planned and actual hours directly.
- Takes distractions into account.
- Gives a clear risk level.
- Shows a friendly tip at the end.
